Lya Code wordmark — 4 estilos para você escolher

Cada opção abaixo renderiza "Lya Code" com uma fonte/tamanho diferente.
Eu recomendo a opção D (Lya.Cloud em 7-row bold) — tem presença visual
máxima, lê bem em qualquer terminal, e respeita o branding do produto
(Lya.Cloud é o domínio canônico, com ponto entre as palavras).

────────────────────────────────────────────────────────────────
Tamanhos disponíveis:
────────────────────────────────────────────────────────────────
- 5-row thin:   ~43 colunas ×  5 linhas  (pequeno, leve)
- 5-row big:    ~53 colunas ×  5 linhas  (médio, padrão)
- 7-row bold:   ~71 colunas ×  7 linhas  (grande, hero)
- 7-row mix:    ~65 colunas ×  7 linhas  (grande + brand)

Texto por linha:
- 5-row thin:  Lya Code visível, mas só metade do tamanho de uma tag
- 7-row bold:  Lya Code ocupa ~70% da largura típica de terminal (80 col)
- 7-row mix:   Lya.Cloud ocupa ~80% da largura típica

Recomendação: D (7-row bold + Lya.Cloud com ponto)
