Privacy Policy

Last updated: June 22, 2026  ·  Studio CodeAI

LyaCode ("the App", "we", "our") is a command-line agentic terminal developed by Studio CodeAI. This Privacy Policy explains what information the App accesses, what it does not collect, and how your data is handled.

1. Information We Do Not Collect

LyaCode does not:

  • Collect or transmit personal information to Studio CodeAI servers.
  • Track usage analytics, crash reports, or telemetry.
  • Store your source code, file contents, or conversation history remotely.
  • Sell, share, or monetize any user data.

2. Local Data

LyaCode stores configuration and session data exclusively on your local machine in the following locations:

  • ~/.lyacode/ — provider profiles, API key references, session logs.
  • %LOCALAPPDATA%\lyacode\ — install logs (Windows).

This data never leaves your device unless you explicitly configure a provider (e.g. Anthropic, OpenAI, Ollama) and send a request through that provider.

3. Third-Party AI Providers

When you configure an AI provider (Anthropic Claude, Google Gemini, OpenAI, Hugging Face, GitHub Copilot, Ollama, Groq, OpenRouter or others), your prompts and context are sent directly from your machine to that provider's API endpoint. Studio CodeAI does not intermediate, log, or inspect this traffic.

Each provider's own privacy policy governs how they handle your data:

  • Anthropic: anthropic.com/privacy
  • Google: policies.google.com/privacy
  • OpenAI: openai.com/privacy
  • Hugging Face: huggingface.co/privacy
  • GitHub: GitHub Privacy Statement

4. OAuth Authentication

If you use the /onboard-github or /onboard-huggingface commands, LyaCode opens your system browser and redirects you to the provider's OAuth flow. Access tokens obtained are stored locally on your device and are used solely to authenticate API requests on your behalf.

5. MCP Servers

LyaCode supports Model Context Protocol (MCP) servers. If you connect an external MCP server, data may be shared with that server according to its own privacy policy. Studio CodeAI does not control or monitor third-party MCP servers.

6. Children's Privacy

LyaCode is a developer tool not directed at children under 13. We do not knowingly collect information from children.

7. Changes to This Policy

We may update this Privacy Policy from time to time. The "Last updated" date at the top of this page reflects the most recent revision. Continued use of the App after changes constitutes acceptance of the updated policy.
